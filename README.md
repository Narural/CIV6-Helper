# CIV6 Helper - Сервис для помощи новичкам в игре Sid Meier’s Civilization VI

## Обзор

CIV6 Helper — это веб-приложение, созданное как вспомогательная платформа для игроков игры Sid Meier’s Civilization VI. Оно предоставляет инструменты и ресурсы для улучшения игрового процесса, включая драфт наций, калькулятор скидок на районы, справочник лидеров с готовыми гайдами на стандартные открытия, возможность добавлять свои билды и форум для обсуждения, включая гайды на научную и культурную победы.

## Особенности

- **Драфт наций**: Инструмент для генерации драфта наций для соревновательной игры.
- **Калькулятор скидок на районы**: Расчет скидок на районы.
- **Список лидеров с гайдами**: Справочник лидеров, каждый с подготовленными гайдами на стандартные открытия.
- **Пользовательские билды**: Авторизованные пользователи могут добавлять и делиться своими билдами, мнением на конкретные билды.
- **Форум для обсуждений**: Коммьюнити-форум для обсуждения билдов, баланса и лидеров

## Типы пользователей

Приложение поддерживает три типа пользователей:
- **Гость**: Ограниченный доступ, может просматривать публичный контент, но не может добавлять или редактировать.
- **Авторизованный пользователь**: Полный доступ для добавления собственных билдов и участия в форуме.
- **Админ**: Полный административный контроль, удаление, редактирование, блокировка и разблокировка пользователей.
